# Movie Recommendation System
This Project is a Movie Recommendation System using Suprise Library in Machine Learning. A Django Application is created for recommendations using the trained model of surprise library. It provides personalized movie recommendations based on user preferences.
## Quick Start
Install required libraries
```
pip install pandas, scikit-surprise
```
Install `Django`
```
pip install django
```
Run 
```
python manage.py makemigrations
python manage.py migrate
```
Run Server
```
python manage.py runserver
```
